# Magna Locações 2026

Repositório oficial do site da **Magna Locações**.

## Projeto
O código da aplicação está em:

- `magnaloc-website/`

## Executar localmente
```bash
cd magnaloc-website
npm install
npm run dev
```

## Build de produção
```bash
cd magnaloc-website
npm run build
```
